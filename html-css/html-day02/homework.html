关于远大

我们必须学会说出自己真正需要的是什么，而不是感觉到什么。如果你优柔寡断，就不要老说“我做决定很痛苦”，而是说“我行事果断”；如果你想减肥，就不要一遍又一遍地说“我管不住自己的嘴，一旦吃上就停不下来”，坚持说自己现在怎样，那就会永远这样。
